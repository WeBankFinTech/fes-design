<h1 align="center">Fes Design</h1>

<div align="center">

一个 vue3 组件库。
</div>

## Install

```bash
npm i @fesjs/fes-design
```

```bash
pnpm i @fesjs/fes-design
```

## Usage

```js
<template>
    <FButton type="primary">Primary</FButton>
    <CloseCircleFilled />
</template>

<script setup>
import {FButton} from '@fesjs/fes-design';
import {CloseCircleFilled} from '@fesjs/fes-design/icon';
</script>
```
